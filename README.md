# Простой "симулятор торговли"
Программа написана с использованием фреймворка Qt. Берет данные с биржи Bybit, строит график цены, некоторых индикаторов, проводит небольшой простенький анализ и дает совет покупать/продавать. Есть возможности симулировать открытие длинных и коротких позиций, устанавливать Take Profit и Stop Loss.
